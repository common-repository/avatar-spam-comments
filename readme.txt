== Description ==

Avatar Spam Comments de-spams comments with avatars that are marked as spam, because they are almost always legitimate comments.

== Installation ==

Either:

1. Upload "avatar-spam-comments.zip" to the "/wp-content/plugins/" directory
2. Activate the plugin through the "Plugins" menu in WordPress

Or:

1. Search for the plugin from within your WordPress Dashboard and install it there.
